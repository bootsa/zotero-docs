# Zotero Settings

**Note:** This section hasn't been updated for Zotero 7, so the screenshots will look different. Most of the settings are unchanged, however.

## Preferences Window

Many of Zotero's features can be customized via the Zotero preferences window. Open the preferences by clicking "Edit → Settings" (Windows/Linux) or "Zotero → Settings" (Mac). You can also press `Ctrl/Cmd`-`,`.

![](/_media/preferences_tabs.png){ width=600 }

The Preference window is divided into the following panes:

-   **[General](preferences/general)**: Adjust appearance, import settings, and other general features.
-   **[Sync](preferences/sync)**: Set up data and file syncing.
-   **[Search](preferences/search)**: Manage PDF fulltext indexing and see relevant statistics.
-   **[Export](preferences/export)**: Set default settings for generating bibliographies and citations.
-   **[Cite](preferences/cite)**: Add, remove, edit, and preview citation styles and install word processor plugins.
-   **[Advanced](preferences/advanced)**: Zotero data location, library lookup, and other advanced settings.

## Hidden Preferences

In addition to the settings shown in the Zotero settings window, Zotero has a number of [hidden preferences](preferences/hidden_preferences) that only can be changed by opening the Config Editor under the Advanced tab of the settings window.
