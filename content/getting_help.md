# Getting Help

Need help with Zotero? To resolve your issue as quickly as possible, please follow the steps below.

## Step 1: Update Zotero

[Upgrade Zotero](installation#how_do_i_upgrade_to_a_new_version) if you aren't running the [latest version](changelog), as your issue may have already been resolved in the most recent release.

## Step 2: Check the troubleshooting pages

If you have a general question about Zotero, check the [Frequently Asked Questions](frequently_asked_questions) and [Knowledge Base](kb).

If you're experiencing a problem, the following dedicated troubleshooting pages may be helpful:

-   [Issues installing Zotero](installation#troubleshooting_installation_issues)
-   [Issues with Zotero data](zotero_data) (e.g., library missing, restoring from a backup)
-   [Issues saving from websites](troubleshooting_translator_issues)
-   [Issues with word processor plugins](word_processor_plugin_troubleshooting)
-   [Issues with data syncing](kb/changes_not_syncing)
-   [Issues with file syncing](kb/files_not_syncing)
-   [Issues with file handling](kb/file_handling_issues) (e.g., PDFs opening in the wrong application)

## Step 3: Post to the Zotero Forums (Really! It Works!)

If you can't find the answer to your question in the documentation, post a message to the Zotero Forums, where you'll get fast, expert support directly from Zotero developers as well as long-time community members. See [How Zotero Support Works](zotero_support) for more information on how this allows us to provide the best possible support.

In general, please start a new thread for your issue. It is much better to start a new thread rather than posting to thread with a potentially unrelated issue. The Zotero developers or volunteers on the forums can point you to a relevant thread if one exists.

If you're new to the Zotero Forums, please read the [forum guidelines](forum_guidelines) and [bug reporting procedures](reporting_problems) before posting.

**Note:** You need to [create a Zotero account](/user/register/) and log in to post messages to the Zotero Forums. You can choose a different username for the forums from your [account settings](/settings/account).
