#### Why don't I see a Zotero icon in the address bar when viewing a webpage?

If you've used Zotero for Firefox previously and no longer see a Zotero save icon in the Firefox address bar, note that the button is now always present in the Firefox toolbar. See [here](adding_items_to_zotero#web_translators) for more details.
