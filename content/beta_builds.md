# Zotero Beta Builds

If you want to run the latest prerelease version of a Zotero component (e.g., to take advantage of a recent bug fix or test a new feature), you can install a beta build. These versions are built regularly and may be less stable than release versions.

(If you want to make code changes, see [Zotero Source Code](dev/source_code).)

## Zotero Beta

**Don't want to run a beta? Download the [release version of Zotero 9](/download/).**

Beta versions of Zotero are currently built from the development line for Zotero 10.

-   [Mac](/download/client/dl?platform=mac&channel=beta)
-   [Windows 64-bit Installer](/download/client/dl?platform=win-x64&channel=beta)
-   [Windows 64-bit ZIP](/download/client/dl?platform=win-x64-zip&channel=beta)
-   [Windows ARM64 Installer](/download/client/dl?platform=win-arm64&channel=beta)
-   [Windows ARM64 ZIP](/download/client/dl?platform=win-arm64-zip&channel=beta)
-   [Windows 32-bit Installer](/download/client/dl?platform=win32&channel=beta)
-   [Windows 32-bit ZIP](/download/client/dl?platform=win32-zip&channel=beta)
-   [Linux 64-bit](/download/client/dl?platform=linux-x86_64&channel=beta)
-   [Linux ARM64](/download/client/dl?platform=linux-arm64&channel=beta)
-   [Linux 32-bit](/download/client/dl?platform=linux-i686&channel=beta)

Beta versions will update automatically. You can update to the latest version at any time via Help → Check for Updates.

To revert to the release version, reinstall it from the [download page](/download), though note that beta versions occasionally update the database such that downgrading isn't possible without reverting to a backup copy of zotero.sqlite in the [Zotero data directory](zotero_data) or removing the local data directory and syncing to pull down data from the online library. You'll always be able to switch to the next major production release of Zotero.

On Windows, you may wish to use the ZIP version, which doesn't contain an installer, to avoid overwriting your primary installation. (Your existing Zotero database will still be used.) Note that the ZIP version does not register Zotero as a handler for various file types.

## Zotero Connector Beta

The Zotero Connector Beta is used for testing connector-specific features. It's not necessary to use a beta version of the connector when using the Zotero Beta above.

### Firefox

-   [Zotero Connector for Firefox (beta)](/download/connector/dl?browser=firefox&channel=beta)

Beta versions will update automatically, or you can update to the latest version at any time from the Firefox Extensions pane. To revert to the release version, simply reinstall it from the [download page](/download).

### Safari

A beta version of the Zotero Connector is included in the [Zotero beta](#zotero) above.

### Chrome/Edge

There is no beta connector for Chrome or Edge.

## Zotero for iOS Beta

To try the latest beta of the iOS app, join the TestFlight group from your iOS device:

<https://testflight.apple.com/join/Rjyua6Mq>

You can switch back to the release version at any time by reinstalling it from the App Store.

We ask that you remove yourself from the beta group in TestFlight if you switch back to the release version, as beta slots are limited.

## Zotero for Android Beta

To try the latest beta of the Android app, first [install Zotero for Android from the Play Store](https://play.google.com/store/apps/details?id=org.zotero.android), and then follow these steps:

1.  Open the Play Store app.
2.  At the top right, tap the Profile icon.
3.  Tap “Manage apps & devices” and then “Installed”.
4.  Tap on Zotero to open its details page.
5.  Under “Join the beta”, tap “Join” and then “Join”.
