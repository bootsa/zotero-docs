**Note:** This page used to hold an updated version of Adam Crymble's [How to Write a Zotero Translator](http://niche-canada.org/member-projects/zotero-guide/chapter1.html). The guidance included in both Adam's original guide and this update is too outdated to be useful and translators based on it are no longer accepted. We instead recommend the following resources:

-   The template code included in [Scaffold](https://www.zotero.org/support/dev/translators/scaffold),
-   The code of [recently updated translators](https://github.com/zotero/translators/pulls),
-   The documentation for writing Zotero translators by [Wikimedia](https://www.mediawiki.org/wiki/Citoid/Creating_Zotero_translators), the most comprehensive and up-to-date resource for writing "scrapers" as of November 2017,
-   The [translator coding documentation](https://www.zotero.org/support/dev/translators/coding),
-   Search for answers in the [Zotero Forums](https://forums.zotero.org/discussions) and [on GitHub](https://github.com/zotero/translators/issues).
