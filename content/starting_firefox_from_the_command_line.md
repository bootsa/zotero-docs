1\. Go to <about:config> in the URL bar, type zotero.debug in the Filter box, and make sure extensions.zotero.debug.log is set to true. (You can change the setting by double-clicking on the line.)

2\. Close Firefox, and then restart it via the command prompt:

Windows:

a\) Start, Run

b\) "C:\\Program Files\\Mozilla Firefox\\firefox.exe" -console

Mac:

a\) Open Terminal in /Applications/Utilities

b\) /Applications/Firefox.app/Contents/MacOS/firefox

You should see debug output in the console as Firefox starts.
